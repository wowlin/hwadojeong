# Orientation Contract

Use this rule for every answer and every code change.

- The drawing is read with the entrance/deck at the bottom.
- Bottom = front / yard / entrance.
- Top = rear wall / stair-back wall.
- Plan-left = family room side.
- Plan-right = living+kitchen side.
- Do not reinterpret plan-left/plan-right by camera angle or viewer position.

When the user says "left" or "right", use the drawing on-screen with the entrance/deck at the bottom.
