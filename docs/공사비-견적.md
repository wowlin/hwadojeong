## 단가 가정 메모
- 철골 30만원/평 — 사용자 제시값. 골조 = **(주)세움스틸하우스(1544-2909)** 단일 업체로 집+썬룸 **통합 제작·시공 권장**(접합부 단일 책임·통합 견적·1회 양중). 썬룸은 노출 도장 스틸이라 부재 형식은 다를 수 있으나 한 업체가 일괄 코디.
- 기초 — **KC금강 시스템말뚝기초**. 말뚝 1본 30만 + 설치비 200만(사용자 제시값). 본수 33 = 현재 3D 모델 기준(집 18·데크 12·썬룸 3). 지반·말뚝 길이 따라 본수/단가 변동.
- 외벽 자재 단가 — [외벽-자재비.md](외벽-자재비.md) 참조(아이소핑크 출처 포함).
- 지붕 — **리얼징크(티타늄아연) 확정**, 시공포함 ~45만/평(통상 개산, 검색 미확보 → 견적 확인 필요). 견적서에 "리얼징크(티타늄아연)"인지 명시 확인(갈바륨 강판을 리얼징크라 파는 경우 많음).
