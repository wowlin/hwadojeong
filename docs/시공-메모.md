# 시공 메모 (화도정) — 잊지 말 것

## 다락 분리(냉난방 효율)
- 1층이 개방형(거실↔현관·계단 한 공간)이라, 분리문은 **계단 하단이 아니라 다락 입구(계단 위)** 에 단다 → 1층 개방 유지 + 다락만 분리. (모델 반영됨)
- 문은 **단열·기밀(웨더스트립)** 사양이라야 효과. 일반 문이면 의미 적음.
- 최대 효율 원하면 **다락 바닥에도 단열** 추가 → 완전 분리(선택).
- 그 위치 다락 천장 ~1.87m라 문 높이 1.8m.


## 지붕: 탄탄보드 단열 시 피스/방수 처리

**핵심:** 탄탄보드는 발포보드(밀도 35)라 **피스를 못 잡는다.** 징크·눈막이·태양광은 탄탄보드가 아니라 **합판 데크 또는 구조(서까래)** 에 고정해야 한다.

### 지붕 구성 (구조 → 마감)
```
스틸 서까래/중도리
  └ 탄탄보드(단열)
      └ 구조용 합판/OSB 데크   ← 피스 잡는 층(탄탄보드 관통해 서까래에 고정)
          └ 자착식 방수시트     ← 피스 구멍을 자가 실링
              └ 리얼징크(스탠딩심)
```

### 피스 자국 처리 — 3가지
1. **리얼징크(스탠딩심):** 클립이 합판에 고정, 그 밑 **자착식 방수시트가 피스 구멍 자가실링**. 마감면엔 피스 안 보임(솔기로 덮임).
2. **태양광·눈막이 = 무피스 "심 클램프"(S-5! 류) ★ 베스트:** 스탠딩심 솔기에 물려 조여 고정 → **지붕에 구멍 0, 누수 0.** 태양광은 풍압 커서 특히 이 방식.
3. **불가피하게 관통 고정 시:** 방수 플래싱/고무부츠 + EPDM 와셔 + 부틸/우레탄 씰링, **반드시 서까래·중도리에 앵커**. "피스+실리콘"만으론 5~10년 뒤 누수.

### 결론
탄탄보드 위 **합판 데크 + 자착식 방수시트** → 피스는 합판이 잡고 구멍은 시트가 막음. **태양광·눈막이는 무피스 심 클램프**로 지붕에 구멍 안 내는 게 50년 가는 정답.
